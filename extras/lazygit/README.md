# Bamboo for Lazygit

## Bamboo Light

Edit `lazygit/config.yaml` and set theme colors like:

```yaml
gui:
  theme:
    activeBorderColor:
      - "#1745d5"
      - bold
    inactiveBorderColor:
      - "#838781"
    optionsTextColor:
      - "#1745d5"
    selectedLineBgColor:
      - "#eaead0"
    cherryPickedCommitBgColor:
      - "#dadac2"
    cherryPickedCommitFgColor:
      - "#177fff"
    unstagedChangesColor:
      - "#c72a3c"
    defaultFgColor:
      - "#3a4238"
    searchingActiveBorderColor:
      - "#a77b00"
```
